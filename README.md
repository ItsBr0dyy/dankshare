DankShare
=====
A image uploader using [Kappa](https://kappa.lol) CDN.

# What Does This Include?

- This includes an **API** for ShareX, Chatterino, DankChat, and your own client. [API Uploaders](https://dankshare.itsbr0dyy.dev/api/upload)
- API enpoints include /api/upload

# Installing
```bash
$ git clone https://github.com/itsbr0dyy/dankshare.git
npm install
```
# Development
```bash
node server.js
```
