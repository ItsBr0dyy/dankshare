API Endpoints

• POST https://dankshare.itsbr0dyy.dev/api/upload   → Upload any file using multipart/form-data   → Key: image

• GET /view/:id   → Access uploaded file

Response Format 
{
  "url": "https://dankshare.itsbr0dyy.dev/view/abc123"
}
Chatterino Integration

• Open Chatterino Settings • Go to External Tools • Set Image Uploader to Custom • Paste this command: 
POST https://dankshare.itsbr0dyy.dev/api/upload image {url}
